# Transcript AI

## Requirements

- Python 3.11
- ffmpeg

## Installation

Create a virtual environment and activate it:
```bash
python3 -m venv .venv
source .venv/bin/activate
```

Install the requirements:
```bash
pip install git+https://github.com/m-bain/whisperx.git torch flask
```

## Usage

```bash
python3 main.py
```
